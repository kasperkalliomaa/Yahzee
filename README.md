# Mini-Yahtzee (Expo / React Native)

- Mobiilipeli, jossa heitetään noppia, lukitaan valintoja ja valitaan pisteet (1–6) yahzin säännöillä.
- Tulokset tallennetaan paikallisesti ja näytetään Scoreboardissa.
- Toteuttettu kouluprojektina.

## Demo
 - https://youtube.com/shorts/1j0RiDxNj74

## Teknologiat
- Expo + React Native
- React Navigation (Bottom Tabs)
- AsyncStorage

## Käynnistys
### Vaatimukset
- Node.js
- npm
- Expo Go

### Asennus & käynnistys
```bash
git clone https://github.com/kasperkalliomaa/Yahzee.git
npm install
npx expo start
